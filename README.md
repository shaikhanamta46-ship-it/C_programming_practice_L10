# C File I/O Programs

This repository contains beginner-friendly C programs demonstrating the basics of **File Input/Output (File I/O)**. These examples cover reading, writing, appending, and processing files using standard C file handling functions.

## Programs Included

* Reading integers from a file using `fscanf()`
* Checking if a file exists before reading
* Writing data to a file using `fprintf()`
* Appending data to an existing file
* Writing a single character using `fputc()`
* Reading a file character by character using `fgetc()`

## Concepts Covered

* `fopen()`
* `fclose()`
* `fscanf()`
* `fprintf()`
* `fgetc()`
* `fputc()`
* File modes: `"r"` (Read), `"w"` (Write), `"a"` (Append)
* Basic file error handling using `NULL`

## Learning Outcome

After completing these programs, you'll understand how to:

* Open and close files safely.
* Read and write data using different file functions.
* Append new content without overwriting existing data.
* Read a file one character at a time.
* Handle file opening errors gracefully.
